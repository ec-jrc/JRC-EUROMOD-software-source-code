Inequality and poverty Tables

3.1. Inequality

Description

The Table shows the following indices:

  1. Gini coefficients for a list of income definitions
  2. Redistribution index: Gini of the first component of the list(default: original income) minus Gini of the last component of the list (default: disposable income)
  3. Social welfare index computed with the last component in the list (default: equivalised disposable income)
  4. Income quantile share ratio
  5. Inter-decile ratio
  6. Atkinson inequality index
  7. Mean log deviation
Note that (1), (2) and (3) refer to incomes as defined by the income definitions list (see arguments), whereas (4), (5), (6) and (7) refer to the last component of the list (default: disposable income, as defined by the income list ils_dispy, and are thus consistent with Table 3.5. and 3.6.).

Note also that negative incomes are recoded to zeros for the calculations of (1), (2) and (3), to keep consistency with the Statistics Presenter.

Note finally that all the indicators are calculated within each scenario, with no connection to the other scenarios. This means, for example, that the income quantiles of (4) are computed independently for each scenario and do not necessarily include the same observations as the baseline or other reform scenarios.

Note that negative incomes are recoded to zero.
This is different from Table 3.2. Progressivity to be consistent with the standard EUROMOD Statistics Presenter.

Arguments
  • Income definitions:
    For details see Inequality and poverty Tables / General argmuments / Income definitions
  • Income quantile share ratio ST/SB:
    Users can define the respective top and bottom quantile (10, 20, ...., 80, 90). Note that the income quantiles are computed within each scenario and do not necessarily include the same observations as the baseline or other reform scenarios.
    Default: T=80, B=20
  • Inter-decile ratio DT/DB:
    Users can define the number of the respective top and bottom decile cut-off (1 - 9)
    Default: T=5, B=1
  • Atkinson inequality aversion:
    Users can define the inequality aversion parameter
    Default: 0.25
