Distributional settings

Changing Table captions

The input boxes starting with Title ... allow for changing the captions of the Pages and Tables displayed by In-depth.

Changing population breakdown

The Breakdown-list allows for defining one or more breakdown specifications.
Defining more than one breakdown specification multiplies the number of Distributional Pages.
Thus, if the list specifies two breakdowns, for example one for quintiles of disposable income and one for quintiles of original income, the tool will generate two Distributional Pages: in the first all Tables will be broken down by quintiles of disposable income, in the second all Tables will be broken down by quintiles of original income.
The button and subtitle of the Page will show the name or description of the Page's breakdown variable.

The buttons alongside the Breakdown-list allow for adding and removing breakdown specifications. Moreover, specifications can be sorted for defining the order of appearance of the resulting Distributional Pages.

Each breakdown specification provides the following options:

  • Row title: a custom title for the breakdown rows
    The respective value of the breakdown variable will be added to this title, thus the Row title ‘Decile’ yields rows ‘Decile 1’, ‘Decile 2’, ... ‘Decile 10’.
    If not provided, the rows will be captioned by the respective values of the breakdown variable only (e.g. 1, 2 ... 10).
    Also see Value descriptions below.
  • Variable: a variable that must exist in the baseline output file
    The variable can be either
    • used as base for building quantiles or
    • its discrete values can be used as the breakdown values
  • Equiv: if ticked, the modified OECD equivalence scale will be applied on the Variable before building quantiles
    This option is ignored if the option Quantiles is not used (see below).
  • Quantiles: number of quantiles the population should be broken into, based on Variable
    If set to n/a or left empty, the discrete values of Variable are used as the breakdown values.
  • Value descriptions: a set of descriptions that will be matched with the discrete values of Variable
    Clicking the button Edit allows for specifying the descriptions.
    The descriptions will replace the actual value in the Row title (see above). If no description is filled in, the actual value is used.

There are five default breakdowns: Disposable_Income_Deciles, Standard_Labour_Categories, Standard_HH_Categories, Standard_Gender_Categories, Standard_Age_Categories. For more details on the default breakdowns see the Default Breakdowns page.

Specifying Variables/Formulas for which to generate Tables

Tables 2.1. Taxpayers-beneficiaries, 2.2. Total tax or benefit and 2.3. Mean tax or benefit provide a Variables/Formulas-list, allowing for the specification of variables or formulas for which to generate the respective Table.
Defining more than one variable/formula multiplies the number of instances of the respective Table.
Thus, if the list specifies two variables/formulas for Table 2.2., for example ils_pen and ils_dispy - ils_pen, the tool will generate two instances of Table 2.2.: one showing totals for ils_pen and the other showing totals for ils_dispy - ils_pen.
The subtitle of the resulting Tables will show the variable/formula (i.e. ils_pen and ils_dispy - ils_pen respectively).

The buttons alongside the Variables/Formulas-list allow for adding and removing variables or formulas. Moreover, rows can be sorted for defining the order of appearance in the Distributional Page.

Changing the level of analysis

All Tables provide a selection-box for the table's Level of analysis, with the options Household and Individual.
For Table 2.1. Taxpayers-beneficiaries the box provides a further option Individuals in household: see General arguments / Level of analysis for the meaning of this option.
For Table 2.6. Mean equivalised disposable income the box is fixed to the option Individual for reasons of clearness.
