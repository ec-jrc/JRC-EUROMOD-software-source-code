Fiscal settings

Changing Table captions

The input boxes starting with Title ... allow for changing the captions of the Pages and Tables displayed by In-depth.

Changing arguments for Table 1.2.

The fields in the group Table 1.2. allow for configuring the content of Table 1.2.
For details see Fiscal Tables / 1.2. Number of taxpayers/beneficiaries.

Changing arguments for Tables 1.3. & 1.4.

The table in the group Table 1.3. & 1.4. defines the so called ‘disaggregated concepts’.
The buttons alongside the table allow for adding, deleting and changing the order of rows, where the latter specifies the order of rows in the Tables displayed by the tool.
Each row configures a row in the displayed Table, and allows for analysing variables and ‘concepts’ (i.e. formulas), which are not covered by the standard Tables.

Example:
Variable/Formula: ils_earns
Formula filter: dgn==0
Payers/Receivers: !=0
Level of analysis: Individual
The respective row in Table 1.3. will show total annual earnings of women.
The respective row in Table 1.4. will show the number of women with non-zero earnings.

For more details see Fiscal Tables / 1.3. Disaggregated concepts and Fiscal Tables / 1.4. Disaggregated concepts, number of units.

The check boxes Bold and Bottom border allow for showing a row in bold letters and displaying a bottom border underneath the row respectively.

Please note that the columns Payers/Receivers and Level of analysis only apply to Table 1.4..

Reset

Clicking the button Reset resets all fields of the dialogue back to their default values.
