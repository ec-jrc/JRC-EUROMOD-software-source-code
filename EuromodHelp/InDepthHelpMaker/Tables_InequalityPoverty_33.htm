Inequality and poverty Tables

3.3. AROP rates

Description

The Table shows FGT(0) at-risk-of-poverty rates for a list of income definitions (see arguments) and a list of poverty lines (see arguments).

Arguments
  • Income definitions:
    For details see Inequality and poverty Tables / General arguments / Income definitions
  • Poverty lines:
    Users can specify the list of poverty lines applied
    Default: 3 poverty line specifications:
    • 60 % relative anchored (i.e. 60% of the individual median of the respective income in the baseline scenario, equivalised using the modified OECD equivalence scale)
    • 50 % relative anchored
    • 40 % relative anchored
    For definition see Inequality and poverty Tables / General arguments / Poverty lines
