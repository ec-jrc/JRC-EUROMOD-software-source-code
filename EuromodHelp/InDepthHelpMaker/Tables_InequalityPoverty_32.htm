Inequality and poverty Tables

3.2. Progressivity

Description

The Table shows the following indices:

  • A = Gini gross income
  • B = Gini net income
  • C = Kakwani index
  • D = Net average tax rate
  • E = Re-ranking effect
  • A - B = C*D - E = Reynolds-Smolensky index (Kakwani decomposition)

Note that negative incomes are not recoded to zero.
This is different from Table 3.1. Inequality which aims to be consistent with the standard EUROMOD Statistics Presenter.

Arguments
  • Gross income:
    Default: ils_origy + ils_ben
    For details with respect to definition see General arguments / Formulas
  • Net income:
    Default: ils_dispy
    For details with respect to definition see General arguments / Formulas
