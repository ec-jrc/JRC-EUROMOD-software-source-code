The special function Scale

The function Scale allows the user to scale monetary variables and monetary parameters.

This function is only available with EM3.

Example 1:
Policy Grp/No SL_demo Comment
Scale on
FactorVariables 1.5

In the above example, all monetary variables will be multiplied by 1.5.

Example 2:
Policy Grp/No SL_demo Comment
Scale on
FactorVariables 2
FactorParameter 2

In example 2, all monetary variables, as well as all monetary parameters will be multiplied by 2.
