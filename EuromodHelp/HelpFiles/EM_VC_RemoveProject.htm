Remove Project

To remove a project from the version control system, press the Remove VC Project button in the ADVANCED OPTIONS group of the VC ribbon. Note that for this to work, you need to be already logged into Version Control and you need to have the required rights to remove projects. Pressing the Remove VC Project button opens the Version Control - Remove Project dialog. This lists all projects currently available on the version control system. By selecting a project and pressing OK the respective project is completely and irrevocably removed from the version control system.

Note that the operation does not affect the local project (the project loaded by the user interface), even if this local project is linked to the just removed remote project.
