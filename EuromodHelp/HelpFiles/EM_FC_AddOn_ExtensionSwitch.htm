The special function AddOn_ExtensionSwitch

The function AddOn_ExtensionSwitch allows you to switch an Extension on or off. This is very useful if you know that a specific Add-on cannot work with a specific Extension, or needs another Extension to be on in order for it to work correctly.

These function has priority over default-switches and switches set via the user interface. Also note that several extensions can be switched with a single AddOn_ExtensionSwitch function, i.e. there can be groups of Extension_Name|Extension_Id/Extension_Switch/[Dataset]/[System].

The function must be used in the ao_control* policy of the add-on.

This function is only available with EM3.

Also see EUROMOD Extensions.
