Summary of parameters for function Elig

Is most frequently used for determining the eligibility for receiving benefits. However, it also allows for determining the liability for paying taxes, as well as evaluating other conditions.

Function Specific Parameters

Name Type Compulsory Unique Description
Elig_Cond condition yes yes A combination of conditions: output variable is set to 1 if they are fulfilled, else to 0.
