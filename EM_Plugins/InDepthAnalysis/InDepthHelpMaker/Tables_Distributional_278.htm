Distributional Tables

2.7. Winners & 2.8. Losers

Description

The Tables show number and share of units (e.g. households, see Level of analysis) that fulfil the ‘winner/loser condition’.
A unit is considered winning, if the positive difference between its equivalised disposable income in the respective scenario and its equivalised disposable income in the reference scenario is higher than the Sensitivity threshold (see below).
Vice versa, a unit is considered losing, if the negative difference between its equivalised disposable income in the respective scenario and its equivalised disposable income in the reference scenario is lower than the Sensitivity threshold (see below).
Disposable income is defined by the income list ils_dispy and is equivalised using the modified OECD equivalence scale.
The reference scenario is defined by the option Compare with baseline / previous scenario in the main settings dialogue.

  • Number: total winning/losing units, using weights
  • Share: percentage of winning/losing units with respect to all units (of the group), using weights
Numbers and shares are shown for the whole population, as well as broken down over a breakdown variable (e.g. income deciles, see Breakdown).
Results can be restricted to a specific population group (see Target population).

Arguments
  • Breakdown:
    Default: deciles of individual equivalised disposable income
    For details with respect to definition see Distributional Tables / General argmuments / Breakdown
  • Sensitivity:
    Default: 0
    For details with respect to definition see General arguments / Formulas
    Examples: 100; ils_dispy*0.01
  • Level of analysis:
    Default: Household
    For details with respect to definition see General arguments / Level of analysis
  • Target population:
    Default: empty, i.e. whole population
    For details with respect to definition see Distributional Tables / General argmuments / Target population
