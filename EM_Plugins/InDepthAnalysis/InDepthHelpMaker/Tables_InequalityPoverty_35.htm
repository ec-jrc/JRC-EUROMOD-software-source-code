Inequality and poverty Tables

3.5. AROP rates by type

Description

The Table shows at-risk-of-poverty rates for equivalised disposable income and a specific poverty line (see arguments), over a breakdown of households (see arguments).
Disposable income is defined by the income list ils_dispy and equivalised using the modified OECD equivalence scale.

Arguments
  • Household type breakdown:
    For definition see Inequality and poverty Tables / General arguments / Household type breakdown
  • Poverty line:
    Users can specify the poverty line applied
    Default: 60 % relative anchored (i.e. 60% of the individual median of the respective income in the baseline scenario, equivalised using the modified OECD equivalence scale)
    For definition see Inequality and poverty Tables / General arguments / Poverty lines
