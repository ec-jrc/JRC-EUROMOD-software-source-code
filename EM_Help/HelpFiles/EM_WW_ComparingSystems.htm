Comparing systems

The user interface allows comparing two systems by highlighting the differences with a specific background colour. Respective options are provided by a dialog, which is opened by right clicking the system, which is compared with another one to open the system's context menu and selecting the menu item Compare System with. Note that differences are highlighted in this system.

The dialog allows selecting the comparison system and the colour for highlighting and offers the following options:

  • Expand Differences: If the box is checked, in addition to highlighting parameters, which show different values for the two systems, the functions and policies they belong to are expanded to make differences immediately visible.
  • Align Side by Side: If the box is checked, the comparison system is moved to the right of the system it is compared to, to facilitate checking what the difference is.
  • Clear Current Marker: If the box is checked, the highlighting of a possible former comparison is removed before highlighting the differences of the current comparison. Not checking the box, i.e. keeping the results of a former comparison, and possibly using a different highlighting colour, may be useful to compare the system with more than one comparison system.

Note that any conditional formatting (see Working with EUROMOD - Conditional formatting) is removed before the comparison.

Also note that the highlighting of a comparison is removed, and conditional formatting reinstalled, by any change (e.g. changing a parameter value, adding a function, etc.). Moreover, the system context menu's item Remove Comparison Marker allows removing the highlighting of a comparison without reinstalling the conditional formatting.
