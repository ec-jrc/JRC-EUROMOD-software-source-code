VC Settings

The button VC Settings in the Version Control ribbon's section USER opens a dialog which allows users to change the configuration of the Version Control tool.

The VC settings menu contains two areas: Proxy and Server.

  • Proxy: It allows configuring a proxy (if needed). To do so, click on Use proxy configuration and then fill in the Proxy URL address and Proxy port. Contact your IT department in order to get this information. If you do no need to configure a proxy, select Do not use proxy configuration.
  • Server: It allows selecting the server in which your EUROMOD projects are stored.
