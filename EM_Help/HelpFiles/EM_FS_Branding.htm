Branding

EUROMOD allows for using a different "brand" than "EUROMOD", which intends to give EUROMOD the look of this alternative brand.

A different brand is implemented by adding a folder "Brand" to the root of the EUROMOD File Structure. This folder must contain a file named "BrandingSettings.xml" with the following content:

  • Brand name: This is the only compulsory setting. The name is used instead of "EUROMOD".
  • Background Image: The name of a picture-file (pgn, bmp, gif, jpg) stored in the Brand folder. The picture is used instead of the EUROMOD logo in the user interface.
  • Background Image Layout: Defines how the Background Image is displayed. Possible values are: Zoom (default), Center, Tile and None.
  • Icon: The name of an icon-file (ico) stored in the Brand folder. The icon is used instead of the EUROMOD icon for the user interface.

The example illustrates the possible content of such a settings file:

<BrandingSettings>
    <BrandName>MyBrand</BrandName>
    <BackgroundImage>MyBrandLogo.jpg</BackgroundImage>
    <BackgroundImageLayout>Tile</BackgroundImageLayout>
    <Icon>MyBrand.ico</Icon>
</BrandingSettings>

Once a different brand is used, the text "POWERED BY EUROMOD" appears in the status bar of the user interface.
